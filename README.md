# Sistem Deteksi Kecanduan Game

Aplikasi berbasis web untuk mendeteksi tingkat kecanduan game menggunakan metode Forward Chaining.

## Fitur
- Input data pengguna
- Proses diagnosa kecanduan
- Menampilkan hasil diagnosa
- Login admin

## Teknologi
- PHP
- MySQL
- HTML CSS

## Cara Menjalankan
1. Jalankan XAMPP
2. Import database ke phpMyAdmin
3. Simpan project di folder htdocs
4. Akses melalui browser (localhost)
